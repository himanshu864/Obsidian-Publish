The **Sieve of Eratosthenes** is an efficient algorithm to find all prime numbers up to a given number $n$.

## Algorithm:

---

- It works by iteratively marking multiples of primes starting from $2$, the smallest prime.
- The algorithm eliminates multiples of each prime $p$, ensuring every composite number is marked once.

## Code

---

```cpp
vector<int> sieve(int n)
{
    vector<int> primes;
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;

	// Mark all primes as true and their multiples as false
    for (int i = 2; i <= n; i++)
        if (isPrime[i])
		{
			primes.push_back(i);
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
		}
    return primes;
}
```

## Complexity:

---

- **Time Complexity**: $O(n \log \log n)$
- **Space Complexity**: $O(n)$ (can be optimized to $O(n / 2)$ using only odd numbers)
